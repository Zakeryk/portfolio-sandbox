# Concept Melee

A Three.js recreation of the Super Smash Bros. Melee main menu.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

## Structure

-   `src/main.js`: Entry point, scene setup, post-processing.
-   `src/scene/MenuScene.js`: Manages the 3D menu elements.
-   `assets/`: Contains SVGs and other assets.
